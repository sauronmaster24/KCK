## Running Locally

```bash
git clone https://github.com/mikbuch/django_css.git
```

```bash
cd django_css
```

```bash
pip install -r requirements.txt
```

```bash
python manage.py migrate
```

```bash
python manage.py runserver
```

### Troubleshooting
